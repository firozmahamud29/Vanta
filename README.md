# Vanta Performance Landing Pages

Premium, high-converting SEO landing pages for VANTA. Built with a "Liquid Performance" strictly monochrome aesthetic to match the official [vanta.store](https://vanta.store).

## 🚀 Live Pages
- [Best Underwear for Running](https://firozmahamud29.github.io/Vanta/best-underwear-running.html)
- [Best Running Underwear (Australia)](https://firozmahamud29.github.io/Vanta/best-underwear-running-australia.html)
- [Step One vs VANTA Comparison](https://firozmahamud29.github.io/Vanta/step-one-vs-vanta.html)

## 🎨 Design System
The project follows a strictly monochrome performance aesthetic:
- **Colors:** Pure Black (#000000) and Pure White (#FFFFFF).
- **Typography:** Barlow Condensed (Headings) and Barlow (Body).
- **Aesthetic:** High-contrast, angular, and performance-focused.
- **Components:** SVG-only icons, seamless transitions, and responsive grid layouts.

## 📂 Project Structure
- `/` - Root containing the primary landing pages.
- `/design-system/` - Design tokens and MASTER specification.
- `/* Vanta/` - Source files and regional theme assets.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Vanilla)
- Google Fonts (Barlow)
- SVG Icons (Lucide/Heroicons)
- GitHub Pages Deployment
