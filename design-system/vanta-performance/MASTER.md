# Vanta Performance Design System - MASTER

> **Brand Ethos:** High-contrast, Monochrome, Performance-First.
> **Source:** [vanta.store](https://vanta.store) (Audited April 2026)

## 🎨 Color Palette
| Token | Hex | Usage |
| :--- | :--- | :--- |
| **Primary Black** | `#000000` | CTA Buttons, Announcement Bar, Logo, Primary Text |
| **Pure White** | `#FFFFFF` | Main Background, Header, Card Backgrounds |
| **Performance Gray** | `#F2F2F2` | Subtle section breaks, Hover states, Grid lines |
| **Dark Gray** | `#333333` | Secondary text, Footer headings |

## Typography
| Type | Font Family | Weight | Case |
| :--- | :--- | :--- | :--- |
| **Headings** | `Barlow Condensed` | 700 / 800 | ALL CAPS |
| **Body** | `Barlow` | 400 / 500 | Sentence |

## Component Specs
- **Buttons:** 
    - Background: `#000000`
    - Text: `#FFFFFF`
    - Border-Radius: `0px` (Strictly angular)
    - Hover: Opacity 0.8 / Invert for secondary
- **Cards:**
    - Background: `#FFFFFF`
    - Border: `1px solid #E5E5E5`
    - Shadow: Minimal/Soft (No deep shadows)
- **Announcement Bar:**
    - Background: `#000000`
    - Text: `#FFFFFF`
    - Font: `Barlow Condensed` (Bold)

## Layout Rules
- **Monochrome Superiority:** Do not use brand colors (Red/Blue/Green) unless they are part of a product image.
- **High Contrast:** Black text on White background. White text on Black background for impact sections.
- **Angular Design:** Zero border-radius on primary elements. 2px - 4px on glass cards if used, but prefer strict corners.
